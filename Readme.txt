Project: Air Quality-Based Health Risk Classification with Machine Learning Models

This project uses machine learning to classify health impact levels from very low to very high based on air quality, weather, and health metrics.

Files:
- Project_Air.ipynb: Main Jupyter Notebook containing feature construction, preprocessing, model training, and evaluation.

How to run:
1. Open Project_Air.ipynb in Jupyter Notebook or VS Code.
2. Run all cells from top to bottom.

Requirements:
- Python
- Libraries: pandas, numpy, and scikit-learn
